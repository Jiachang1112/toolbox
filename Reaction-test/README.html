ReactionLab 使用說明

這是一個可離線使用的本地網頁工具，提供多種反應力測試模式：

  • 簡單反應：畫面變綠立刻按下。
  • Go/No-Go：綠點紅不點，考驗抑制能力。
  • F1 反應：模擬起跑燈變綠後起跑。
  • 點擊目標：隨機位置的目標圈，訓練瞄準＋點擊。
  • WASD：看到提示按相對應鍵。

右側提供個人最佳、最近 20 次與成就系統，資料全存於你的瀏覽器 localStorage。

可在下方設定區調整測試次數、延遲範圍、區域大小、輸入方式（滑鼠或空白鍵），並可切換主題、關閉音效。

直接以 index.html 開啟即可離線使用，也可部署到 GitHub Pages / Cloudflare Pages。
